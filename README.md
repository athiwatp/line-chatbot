# Go Line Chatbot Webhook

## How to use

1. Install Golang
```sh
$ brew install golang
```

2. Install dependencies
```sh
$ glide install
```

3. Build binary
```sh
$ go build
```

4. Run binary
```sh
$ ./go-line-chatbot
```
