# Line Chatbot Webhook Written in Go Language

## How to use

1. Install Golang
```sh
$ brew install golang
```

2. Install dependencies
```sh
$ glide install
```

3. Build binary
```sh
$ go build
```

4. Copy `env.sample` to `.env`, and fill environment variables
```sh
$ cp env.sample .env
```

5. Run binary
```sh
$ ./line-chatbot
```
